# 🍽️ Restaurant AI Assistant

An **AI-powered Restaurant Assistant** built with **Node.js, Express.js, LangChain, and Google Gemini 2.5 Flash**. It enables users to interact with restaurant menus using natural language and delivers intelligent, conversational responses.

## 🚀 Features

* 🤖 AI-powered chatbot
* 🍽️ Smart menu lookup
* ⚡ LangChain Tool Calling Agent
* 💬 Natural language interaction
* 🌐 Fast REST API backend

## 🛠️ Tech Stack

* HTML5, CSS3, JavaScript
* Node.js & Express.js
* Google Gemini 2.5 Flash
* LangChain
* Zod

## ⚙️ Setup

```bash
git clone https://github.com/CHANDRASHEKHAR234/Restaurant-AI-Assistant.git
cd Restaurant-AI-Assistant
npm install
```

Create a `.env` file:

```env
GOOGLE_API_KEY=YOUR_GEMINI_API_KEY
```

Run the project:

```bash
npm start
```

Visit: **http://localhost:3000**
